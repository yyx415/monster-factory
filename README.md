# ⚙️ 怪物工厂 (Monster Factory)

![Status](https://img.shields.io/badge/status-Active-brightgreen)
![Language](https://img.shields.io/badge/language-JavaScript-yellow)
![License](https://img.shields.io/badge/license-MIT-blue)

一个轻量级的前端回合制战斗游戏，集合了卡牌合成、策略布阵、冒险挑战等玩法。使用 AI 辅助开发完成。

## 🎮 核心特性

- 🛒 **怪物合成系统** - 通过拖拽两只怪物进行合成，合出更强的怪物
- ⚔️ **策略布阵** - 自由组织出战阵容，不同怪物有不同属性和技能
- 🗺️ **多种游戏模式**
  - 冒险模式：逐关挑战，获得金币奖励
  - 无尽模式：无限波次挑战，考验实力
  - 自由模式：测试场景，自由对战任意组合
- 📋 **完整图鉴系统** - 浏览所有怪物信息和合成公式树
- 💾 **存档管理** - 支持多个存档槽位，随时存档和读档
- 🎨 **深色UI** - 优化的游戏界面，支持移动设备

## 🕹️ 游戏玩法

### 📖 基本概念

| 模块 | 说明 |
|------|------|
| **🧪 实验室** | 购买基础怪物，合成更高级怪物的地方 |
| **⚔️ 队伍** | 组织出战阵容（最多4只怪物） |
| **🗺️ 冒险** | 按顺序挑战关卡，赚取金币 |
| **♾️ 无尽** | 无限波次挑战，考验持久力和策略 |
| **🔬 自由** | 自由组队对战，测试不同组合 |
| **📖 合成树** | 查看所有怪物的合成公式和进化路线 |
| **📋 怪物图鉴** | 浏览所有怪物的详细信息和属性 |

### 💡 游戏流程

1. **购买怪物** → 在实验室的商店用金币购买基础怪物
2. **合成进化** → 拖拽两只怪物到合成槽位，合成更强的怪物
3. **组织队伍** → 在队伍模块选择4只怪物进行布阵
4. **开始冒险** → 选择冒险/无尽/自由模式进行战斗
5. **赢取奖励** → 获得金币，继续购买和合成

### 🎯 核心机制

- **合成消耗** - 合成怪物需要支付金币，不同怪物消耗不同
- **属性克制** - 不同属性怪物有相克关系（目前机制）
- **技能系统** - 各怪物拥有不同技能，如远程、近战、回血等
- **等级系统** - 通过合成提升工厂等级，解锁更多内容

## 🚀 快速开始

### 系统要求
- 现代网页浏览器（Chrome、Firefox、Safari、Edge）
- JavaScript 已启用
- 屏幕宽度建议 ≥ 320px（优化了移动设备）

### 安装与运行

**方式一：直接打开（推荐）**
```bash
# 1. 克隆项目到本地
git clone https://github.com/yyx415/monster-factory.git
cd monster-factory

# 2. 用浏览器打开 index.html
# Windows: 双击 index.html
# Mac/Linux: 右键选择"打开方式" > 选择浏览器
#         或在终端运行：open index.html
```

**方式二：本地服务器（可选）**
```bash
# 如果直接打开出现跨域问题，可用本地服务器
# Python 3
python -m http.server 8000

# 然后访问：http://localhost:8000
```

## 📁 项目结构

```
monster-factory/
├── index.html              # 游戏主页面
├── styles.css              # 全局样式表
├── README.md               # 项目说明文档
├── js/
│   ├── config.js           # 游戏配置（怪物数据、属性等）
│   ├── levels.js           # 关卡和敌人配置
│   ├── state.js            # 游戏状态管理和存档逻辑
│   ├── ui.js               # UI 更新和用户交互处理（核心逻辑）
│   ├── synthesis.js        # 合成系统逻辑
│   ├── canvas-battle.js    # 战斗系统和动画渲染
│   └── main.js             # 程序入口
└── images/                 # 游戏资源文件夹
    ├── mech_*.png          # 机械系列怪物图片
    └── plant_*.png         # 植物系列怪物图片
```

### 文件说明

| 文件 | 大小 | 说明 |
|------|------|------|
| config.js | ~12KB | 怪物模板、属性、技能配置 |
| ui.js | ~46KB | 最大的模块，包含所有 UI 事件和页面更新 |
| canvas-battle.js | ~79KB | 战斗系统，处理动画和战斗逻辑 |
| levels.js | ~13KB | 冒险模式的关卡配置 |
| state.js | ~6KB | 游戏存档和状态持久化 |

## ⚙️ 操作说明

### 基本交互

| 操作 | 说明 |
|------|------|
| **点击购买按钮** | 花费金币购买怪物 |
| **拖拽怪物** | 将怪物从仓库拖入合成槽位或队伍槽位 |
| **点击合成按钮** | 合成两只怪物（消耗金币） |
| **点击怪物卡牌** | 查看怪物详细属性 |
| **切换 Tab 标签** | 在不同模块之间切换 |
| **存档/读档** | 点击顶部 💾 按钮管理存档 |

### 快捷功能

- 💾 **保存** - 保存当前游戏进度到存档槽位
- 📂 **存档管理** - 查看、加载或删除存档
- 🗑️ **重置** - 清空所有进度并重新开始（谨慎使用！）
